# ランログ — 心拍数 × ペース

Apple Watch のランニング記録(心拍数・ペース・距離)を1分ごとにグラフ化する、依存ゼロの PWA です。

## 使い方

1. iPhone の Safari でこのページの URL を開く
2. 共有ボタン → **「ホーム画面に追加」** でアプリのように使えます
3. 「取り込み」タブから、ヘルスケアの書き出しデータ(`export.xml`)を読み込む

### ヘルスケアのデータを書き出す手順

1. iPhone の「ヘルスケア」アプリ → 右上のアイコン → **「すべてのヘルスケアデータを書き出す」**
2. 「"ファイル" に保存」を選択
3. ファイルアプリで zip をタップして解凍 → `apple_health_export/export.xml`
4. アプリの「ファイルを選ぶ」から `export.xml` を選択

## 機能

- 📅 **カレンダー** — ランの日が赤く光る。タップで過去の記録へ
- 📈 **比較** — 任意期間の距離・ペース・心拍の推移と平均(前期間比つき)
- 📊 **詳細** — 1分ごとの心拍 × ペースの2軸グラフ、CSV 書き出し
- 💾 **保存** — 記録は端末内に保存。バックアップの書き出し/復元に対応

## プライバシー

すべてのデータは端末内(ブラウザ)でのみ処理・保存され、どこにも送信されません。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体(HTML/CSS/JS すべて) |
| `sw.js` | Service Worker(オフライン対応) |
| `manifest.webmanifest` | PWA マニフェスト |
| `apple-touch-icon.png` / `icon-512.png` | アイコン |
| `tools/` | 開発・メンテ用スクリプト(アイコン生成・ローカル確認) |

## 開発・引き継ぎの方へ

手元で確認するには `node tools/serve.js` を実行して `http://localhost:8745` を開いてください。
